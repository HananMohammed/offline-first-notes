# offline-first-notes

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
### To implement PWA
```
vue add pwa
```
### To buid output dist of progessive web app 
```
npm run build --production
```
### To serve on php server 
```
cd dist
php -S 127.0.0.1:8000
```
